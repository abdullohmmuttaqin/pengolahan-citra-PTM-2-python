# Pengolahan Citra PTM 2 Python

Project sederhana untuk mempelajari dasar pengolahan citra menggunakan Python, OpenCV, dan Matplotlib.

## Fitur

Program ini melakukan beberapa proses pengolahan citra dasar:

- Menampilkan gambar asli
- Konversi gambar dari BGR ke RGB
- Konversi ke Grayscale
- Gaussian Blur
- Edge Detection menggunakan metode Canny

## Library yang Digunakan

Install dependency dengan perintah berikut:

```bash
pip install opencv-python matplotlib
```

## Cara Menjalankan

Clone repository:

```bash
git clone https://github.com/abdullohmmuttaqin/pengolahan-citra-PTM-2-python.git
```

Masuk ke folder project:

```bash
cd pengolahan-citra-PTM-2-python
```

Jalankan program:

```bash
python main.py
```
